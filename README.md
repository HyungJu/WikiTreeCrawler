#WikiTreeCrawler
**Crawl WikiTree posts from Facebook Page Data**

by WikiRecent Dev Team

---

## Usage
`pip3 install -r requirements.txt`
to Intsall Requirements

`python3 crawl.py` to crawl data

You could customize the number of posts to crawl by modifying the `pages` value in line 6.
